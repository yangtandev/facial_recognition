# Release Notes

## v1.0.2 - 2026-06-29

- 修正遠端同步新增或刪除建檔照後，語音檔不會同步補產或清理的問題。
- 修正齒輪配置介面變更語音文字後，既有 mp3 不會被新內容覆蓋的問題。
- 產生語音檔時改用暫存檔替換，避免 gTTS 失敗時破壞既有語音檔。
- 在辨識畫面上方提示區顯示目前版本號，方便現場回報問題。

## v1.0.1 - 2026-06-23

- 修正報表圖片保留邏輯，強制只保留 7 天內圖片。
- 補上排程共用方向模式，支援更彈性的進出方向設定。

## v1.0.0 - 2026-06-05

- 修正動態模糊人臉仍可能通過品質檢查的問題。
- 調整人臉品質門檻，強化反光、裁切、側臉與模糊畫面的拒絕判斷。
- 優化辨識灰色區間提示、同身份連續辨識間隔與服裝檢查失敗語音提示。
- 將截圖與臉部影像保存流程統一為 PNG，減少重複 JPG 檔案。
